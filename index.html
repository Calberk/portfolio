ABOUT ME

My name is Edmund Park. I am a web developer born and raised in Southern California.

Growing up I enjoyed building things and my curiosity often drove me to want to open up my toys to see how they worked. My first experience with coding was creating a basic html website for an elective course in college. I remember being amazed at seeing the code take shape piece by piece, customized to my design!

After graduating from UC Berkeley my career path led me to working production/operations at a clothing manufacturing company. The fast-paced environment involved having to coordinate between the design, assembly line, technical, vendors and customer service departments. The start-to-finish process allowed me to understand the importance of quality control, working as a team and keeping open communication to achieve success.

I decided to pursue web development because I wanted a career that I was passionate about and enjoyed. I fell in love with coding and it has sparked the same curiosity I had growing up. Now, I look forward to waking up to work on projects and to further my knowledge in all things code. I hope to continue that as I grow my career as a developer and would love to connect with you to learn more.

Bookworms

react with Redux Application

Bookworms is a study tool for students to find and create study groups. After creating accounts users are then able to create groups based on subject, class, location, time and group size as well as search for existing study groups that other users have created. The site is focused on UC Irvine and includes an interactive campus map.

Made using React.js, Redux, JavaScript, Bootstrap, Flexbox with a Node.js and MySQL backend.

Buzzer Eater

Hackathon Group Project using APIs

Buzzer Eater is a web application for users who love basketball and want to go out to eat and enjoy the game.

Initially built in 48hrs during a hackathon with a group of 5. Created using Google Location/Geocoding, Twitter, Zomato, and basketball API's to find information on today's games, live scores, NBA twitter feed, sports bars details and location. Written in JavaScript using API's, AJAX, jQuery and bootstrap.

GitHub Live
